# imOlogy — Portfolio & Coursework

Welcome to the public index of my private coursework and personal projects. This repository serves as a showcase for reviewers and collaborators. Each project below links to a brief overview containing screenshots, tech stacks, and implementation details.

## Projects

- **[BSM103 — Wedding Organization System](./BSM103_WeddingManagement.md)**
  A C++ console application using pipe-delimited text files to model relational data (weddings, guests, costs). Features include file-stream CRUD operations, cascade deletion, and automated seating-plan generation.
  _Tech: C++17, std::fstream._

- **[BSM104 — ahmadMOlogy Personal Portfolio](./BSM104_Portfolio.md)**
  A responsive portfolio website utilizing a modern frontend stack with a PHP-backed login and dashboard system.
  _Tech: Vue.js, Vite, PHP, Responsive CSS._

- **[BSM311 — CLIMAX FITNESS (Gym Management)](./BSM311_GymManagement.md)**
  Full-stack web application featuring class scheduling, administrative analytics, and an integrated AI coaching module.
  _Tech: C#, ASP.NET Core 8, EF Core, PostgreSQL, Gemini AI, Chart.js._

- **[BSM313 — IoT Smart Chessboard](./BSM313_IoT_SmartChess.md)**
  A hybrid Computer Vision and IoT project. Detects moves via OpenCV, processes logic via Stockfish on PC, and visualizes moves on a physical 8×8 LED matrix driven by ESP32.
  _Tech: Python (OpenCV), Stockfish, ESP32 (Arduino), Blynk._

- **[QR Hotel Review System](./QR_Hotel_Review_System.md)**
  A mobile-first static web app allowing guests to write a single review and copy-paste it to multiple platforms via a seamless QR scan workflow.
  _Tech: Vanilla JS, Tailwind CSS, Clipboard API._

- **[SWE417 — SOS Game Tracker](./SWE417_SOS_Game_Tracker.md)**
  Real-time Computer Vision referee for the pen-and-paper "SOS" game. Uses OCR and SVM to log moves, draw HUD overlays, and support model retraining.
  _Tech: C++17, OpenCV, Tesseract, CMake._

- **[TradeScout — AI-Powered Import/Export Intelligence](./trade-scout.md)**
  Market-intelligence prototype combining LLM prompt engineering with search scraping to identify trade leads and map data.
  _Tech: FastAPI, Streamlit, Redis, Supabase, Google Gemini, SerpAPI._
