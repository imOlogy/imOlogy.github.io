# imOlogy — Project Sneak Peek

This repository is a public index and sneak peek of my private coursework and personal projects. Each project below links to a short, read-only overview page (open the link to see details). Full source repositories are private — if you'd like access to any full project, see "Request access" below.

## Projects (quick browse)
- [BSM103 — Wedding Organization System](./BSM103_WeddingManagement.md)  
  C++ console app using pipe-delimited text files to model relational data (weddings, guests, costs). Implements file-stream CRUD, cascade delete, and seating-plan generation. Tech: C++17, std::fstream.

- [BSM104 — ahmadMOlogy Personal Portfolio](./BSM104_Portfolio.md)  
  Responsive portfolio site (HTML/CSS/JS + Vue.js, Vite) with a PHP-backed login/dashboard. Tech: Vue, Vite, PHP, responsive CSS.

- [BSM311 — CLIMAX FITNESS (Gym Management)](./BSM311_GymManagement.md)  
  Full-stack web app (ASP.NET Core 8, EF Core, PostgreSQL) with scheduling, admin analytics, and an AI coaching module (Gemini integration). Tech: C#, .NET 8, EF Core, Chart.js, Flatpickr.

- [BSM313 — IoT Smart Chessboard](./BSM313_IoT_SmartChess.md)  
  Vision + embedded project: OpenCV move detection + Stockfish engine on PC, ESP32 drives an 8×8 LED matrix via Blynk. Tech: Python (OpenCV), Stockfish, ESP32 (Arduino), Blynk.

- [QR Hotel Review System](./QR_Hotel_Review_System.md)  
  Mobile-first static web app that lets guests write a review and post it to multiple platforms via a single QR scan. Tech: Vanilla JS, Tailwind CSS (CDN), Clipboard API.

- [SWE417 — SOS Game Tracker](./SWE417_SOS_Game_Tracker.md)  
  Real-time CV referee for pen-and-paper "SOS" using OpenCV + OCR (Tesseract / SVM). Logs moves, draws HUD overlays, supports retraining an SVM model. Tech: C++17, OpenCV, Tesseract, CMake.

- [TradeScout — AI-Powered Import/Export Intelligence](./trade-scout.md)  
  Market-intelligence prototype that combines LLM prompt engineering with SerpAPI and Google Maps scraping to find trade leads. Tech: FastAPI, Streamlit, Redis, Supabase, Google Gemini, SerpAPI.
